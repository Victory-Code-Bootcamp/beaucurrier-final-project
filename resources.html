This is a webpage to be used for a personal website or portfolio. It defines various styling rules for different elements on the page, including typography, layout, colors, and more. Here's a breakdown of the different sections:

Fonts:
The stylesheet defines two font faces: Reey and Roboto. The Reey font is used for the heading elements, while Roboto is used for the body text.

Glow Effect:
The .glow class is used to create a glow effect on certain elements. It uses the text-shadow property to create a series of shadows that give the impression of a glowing effect. How to do the Glow Effect.

Grid Layout:
The stylesheet defines a grid layout for the page, with different areas for the header, navigation, content, sidebar, ad, and footer. It also defines responsive breakpoints for smaller screens. The Grid I used

Styling:
The stylesheet defines various styling rules for different elements, including: Header: Background color, text color, font size, and padding. Body: Background color, text color, font family, and font size. Navigation: Display, justify-content, and flex-direction. Content: Display, padding, and margin. Sidebar: Display, padding, and margin. Ad: Display, padding, and margin. Footer: Background color, text color, font size, and padding.

Form Styling:
The stylesheet defines styling rules for a contact form, including: Display, flex-direction, and align-items. Label and input field styling. Submit button styling.

Fancy Button:
The stylesheet defines a fancy button style, with a background color, text color, font size, and padding. It also defines hover and focus effects. Examples of cool CSS buttons.

Footer:
The stylesheet defines a grid container for the footer, with different cards for phone, email, and social media information. It also defines responsive breakpoints for smaller screens.

Media Queries:
The stylesheet defines several media queries to adjust the layout and styling for different screen sizes.
